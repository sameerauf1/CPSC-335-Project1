Algorithm 1 - hamilton.py
--------------------------------------------------------------------------------------------------------------------------------
* This algorithm is designed to find the index of the city to start a round trip with given a list of city distances, 
  fuel at gas stations, and the car's miles per gallon that is able to make a round trip without running out of fuel.

* To run the algorithm, make sure you have Python installed on your computer. You can download Python at 
  https://www.python.org/downloads/.

To run Python on VSCode, follow these steps:

* Open VSCode
* Open the hamilton.py file in VSCode
* Click the Run button in the top menu bar
* To test different test cases, modify the values of city_distances, fuel, and car_mpg in the code and run the Python file.
  Make sure to use valid values for these variables. 
* Note the input for this algorithm is: 
* city_distances is a list, representing distances between each city
* fuel is a list, representing fuel available in each gas station
* car_mpg is an integer, representing the carâ€™s mile-per-gallon rate
* The output will be the index(an integer) of the best city to start the round trip with

Algorithm 2 - schedule.py
--------------------------------------------------------------------------------------------------------------------------------
* This algorithm takes two or more arrays as input. The arrays represent availabilities and daily working periods of group 
  members. The output is an array containing intervals of time when all members are available for a meeting.

To run schedule.py, follow these steps:
* Open terminal
* Navigate to directory containing the .py file(s) 
* Execute by typing python3 followed by the filename.
  Ex: python3 schedule.py
* To modify input, replace p1_S, p1_D, p2_S, p2_D, and duration with intended values. Then, execute again.
